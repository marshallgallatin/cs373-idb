Data

Spoonacular

Our recipe data was scraped programmatically from Spoonacular's Food API using Python scripts. The requests package allowed us to send GET requests and parse the responses in JSON format. Here is an example of such a request using a Python script:

#!/usr/bin/env python3
import requests
payload = {'Accept': 'application/json', 'X-Mashape-Key': }
response = requests.get('https://spoonacular-recipe-food-nutrition-v1.p.mashape.com/recipes/search?cuisine=cajun&diet=pescetarian', headers=payload)
data = response.json()
for item in data['results']:
	print(item['title'])

USDA

The US Department of Agriculture maintains an enormous Nutrient Database that is open to the public and even provides a RESTful API to programmatically lookup data. For instance, the following GET invocation will return foods and ingredients that contain the word "butter" in their database:

http://api.nal.usda.gov/ndb/search/?format=json&q=butter&sort=n&max=25&offset=0&api_key=DEMO_KEY
